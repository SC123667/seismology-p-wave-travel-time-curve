# 地信学院地震学作业-P波走时曲线

这是一个基于 PREM 模型计算地球内部 P 波走时曲线的 MATLAB 作业示例。

仓库中包含：

- `prem.txt`：PREM 模型数据文件
- `p_wave_travel_time_homework.m`：MATLAB 主程序
- `运行说明.txt`：运行说明
- `P波走时结果表.csv`：计算结果表
- `图1_P波走时曲线_0到180度.png`：全局走时曲线
- `图2_P波走时曲线_10到35度_三重分支.png`：局部放大走时曲线

程序主要完成以下内容：

1. 读取 PREM 模型中的深度和 P 波速度数据
2. 对球形地球模型做地球扁平化变换
3. 采用 201 个射线参数进行简化射线追踪
4. 绘制题目要求的两张 P 波走时图

适用环境：

- MATLAB

运行方法：

在 MATLAB 当前文件夹进入本目录后，直接运行：

```matlab
p_wave_travel_time_homework
```
